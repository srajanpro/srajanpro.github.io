What is an IIFE in JavaScript?

Posted by Srajan Gupta on February 3rd, 2021

If you are reading this blog post? The first question that should hit your mind is, what's IIFE?

IIFE stands for Immediately Invoked Function Expression

IIFE is a JavaScript function that runs as soon as it is defined.

An IIFE expression is a good way to protect the scope of your function and the variables within it.

If this sounds a bit complicated, don't worry, it's not.
